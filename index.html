Homogeneous Sheared Turbulence

Apr 11th, 2013

Onto turbulence! Let’s take a look at some DNS data from homogeneous sheared turbulence that I analyzed for a class project. We were given DNS results and had to calculate a number of statistics.

First, we’ll import the data, reshape it and calculate min/max to get a feel for the magnitudes we’re dealing with.

We can clearly see the effects of shear in our U velocities. Let’s see how our velocity fluctuates about averages taken over planes perpendicular to the Y axis.

Fluctuating Velocities $u’$, $v’$, and $w’$

hstVmean=zeros(nx[1])
hstWmean=zeros(nx[1])

hstVprime=zeros(nx)
hstWprime=zeros(nx)

for ii in range(0,nx[1]):
    #Means
    hstUmean[ii]=mean(hstU[:,ii,:])
    hstVmean[ii]=mean(hstV[:,ii,:])
    hstWmean[ii]=mean(hstW[:,ii,:])

    #fluctuating component
    hstUprime[:,ii,:]=hstU[:,ii,:]-hstUmean[ii]
    hstVprime[:,ii,:]=hstV[:,ii,:]-hstVmean[ii]
    hstWprime[:,ii,:]=hstW[:,ii,:]-hstWmean[ii]

The default PyLab histogram function creates ugly bar charts, but we want histograms that Kolmogorov would be proud of so let’s make our own histogramOutline function.

In [21]:
def histOutline(dataIn, *args, **kwargs):
    (histIn, binsIn)=histogram(dataIn, density=1,*args, **kwargs)

    stepSize = binsIn[1]-binsIn[0]

    bins = zeros(len(binsIn)*2 + 2, dtype=float)
    data = zeros(len(binsIn)*2 + 2, dtype=float)

    for bb in range(len(binsIn)):
        bins[2*bb + 1] = binsIn[bb]
        bins[2*bb + 2] = binsIn[bb] + stepSize
        if bb < len(histIn):
            data[2*bb + 1] = histIn[bb]
            data[2*bb + 2] = histIn[bb]

    bins[0] = bins[1]
    bins[-1] = bins[-2]
    data[0] = 0
    data[-1] = 0

    return (bins, data)

These results are pretty cool because we can see that the shear makes our $u’$ velocities deviate from a normal distribution while the other velocity fluctation components remain much more normally distributed.

Welcome

Apr 8th, 2013

Minus Five-Thirds

A blog about turbulence and fluid dynamics

Welcome to Minus Five-Thirds, I’m a graduate student studying turbulence and I will be using this blog to document my research. I’m writing this in an IPython Notebook and I intend for this blog to be somewhat of a digital lab notebook where I’ll put scripts, analysis, writings, media, etc. I’m converting the .ipynb files to static HTML using nbconvert, incorporating them into Octopress blog posts using the setup help found here and then posting to Github pages where I can link the blog to code repositories.

I chose to use IPython notebooks because they allow for an interactive PyLab environment (NumPy, SciPy, MatPlotLib, etc) with inline plotting, LaTex math equations, embedded media, and all sorts of other goodies. I’m stoked about the possibilities. For example, everybody’s favorite equations:

\begin{equation} \frac{\partial u_{i}}{\partial t} + \frac{\partial u_{i}u_{j}}{\partial x_{j}} = -\frac{1}{\rho}\frac{\partial p}{\partial x_{i}} +\nu\frac{\partial^2u_{i}}{\partial x^2_{j}} \end{equation}

appear quite readily using $\LaTeX$.

I should also explain this blog’s namesake, “Minus Five-Thirds”, refers to Kolmogorov’s famous 1941 relationship between the energy spectrum $E(k)$, the dissipation rate $\epsilon$, and the wavenumber $k$ given by

\begin{equation} E(k)=C \epsilon^{2/3} k^{-5/3} \end{equation}

but more on that later.

The great thing about using IPythong Notebooks is that once it’s finished, I can share it interactively online (great for collaboration), package it up as a python script, pdflatex it for manuscripts, or convert it to a blog post like I’m doing here. IPython notebooks are a great way to make reproducible science, which is important when you come back to a project after a few weeks and can’t remember where you left off, or if you want to recreate results from a journal publication.

Hosting this as a static page on GitHub will hopefully make for a fast and foolproof website that is well integrated with git version control for keep track of various scripts and software development activities.

Stay tuned for more!
